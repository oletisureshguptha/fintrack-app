# Personal Finance Companion App

## Overview
A mobile app to track daily expenses, income, and savings with a clean and modern UI.

## Features
- User Authentication (Login & Signup)
- Add / Edit / Delete Transactions
- Dashboard with Balance, Income, Expense
- Persistent Login using SharedPreferences
- Modern UI with Gradient & Glassmorphism

## Tech Stack
- Flutter
- Provider (State Management)
- SharedPreferences (Local Storage)

## How to Run
flutter pub get
flutter run

## Assumptions
- Single user per device
- Local data storage (no backend)

## Future Improvements
- Charts & analytics
- Cloud sync
- Notifications
-Otp verification
-Automatic taking inputs
